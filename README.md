# pages-500-lab

## 项目简介

`pages-500-lab` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定的域名或网站，仅作为静态 HTML 资源的集中存放与版本管理空间。所有内容均为独立的页面文件，适合用于测试、演示、学习或轻量级展示用途。

## 目录结构

```
pages-500-lab/
├── pages/          # 存放主要 HTML 页面文件
├── assets/         # 页面引用的静态资源（如图片、样式、脚本等）
└── README.md       # 本文件
```

- `pages/`：各独立 HTML 页面均放置于此目录下，按需可进一步按功能或主题划分子目录。
- `assets/`：页面依赖的外部资源文件，保持相对路径引用，便于本地或在线预览。

## 页面归档说明

本项目中的 HTML 页面为独立归档，不依赖后端服务或数据库。每个页面均可直接通过浏览器打开运行。归档内容可能包括：

- 技术演示页面
- 原型或概念验证
- 学习笔记或文档页
- 其他临时或长期保留的 HTML 资源

页面文件命名应简洁明了，建议以英文或拼音命名，避免特殊字符。
